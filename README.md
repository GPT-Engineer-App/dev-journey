# dev-journey

A personal website for a programmer that showcases their skills, projects, and professional journey. The website will include sections for About Me, Projects, Blog, and Contact Information. It will feature a clean and modern design, with a focus on readability and user experience.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/dev-journey.git
cd dev-journey
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
